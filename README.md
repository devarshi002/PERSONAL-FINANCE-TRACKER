# Personal Finance Tracker

This project is a Python-based personal finance tracker that allows users to manage and monitor their financial transactions. The application can record transactions, view them within a specified date range, summarize income and expenses, and visualize financial data.

## Features

- **Add Transactions**: Record transactions with details such as date, amount, category (Income/Expense), and description.
- **View Transactions**: View transactions within a specified date range.
- **Summary**: Summarize total income, total expenses, and net savings over a given period.
- **Data Visualization**: Plot income and expenses over time using a line graph.

## Installation

### Prerequisites

- Python 3.x
- Pandas
- Matplotlib

You can install the required packages using pip:

```bash
pip install pandas matplotlib
```

### Clone the Repository

Clone this repository to your local machine using:

```bash
git clone https://github.com/devarshi002/PERSONAL-FINANCE-TRACKER.git
```

Navigate to the project directory:

```bash
cd PERSONAL-FINANCE-TRACKER
```

## Usage

### Running the Application

To run the application, execute the `main.py` file:

```bash
python main.py
```

### Options

1. **Add a New Transaction**: Enter details for the transaction.
2. **View Transactions and Summary**: View and summarize transactions within a specified date range.
3. **Exit**: Close the application.

### Example

```bash
1. Add a new transaction
2. View transactions and summary within a date range
3. Exit
Enter your choice (1-3): 1
```

## File Structure

- `main.py`: The main entry point of the application.
- `data_entry.py`: Contains helper functions to get user input for date, amount, category, and description.
- `finance_data.csv`: CSV file to store transaction data.
- `requirements.txt`: List of required Python packages.
